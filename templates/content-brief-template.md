## 4. Content Outline (H2/H3 Structure)

```
# [H1 — Article Title with Primary Keyword]

## [H2 — Section 1: Answer the core question immediately]
### [H3 — Subsection if needed]

## [H2 — Section 2: Context / Background]
### [H3]
### [H3]

## [H2 — Section 3: Deep Dive / How-To / Comparison]
### [H3]
### [H3]

## [H2 — Section 4: Practical Application / What To Do]
### [H3]

## [H2 — Section 5: Common Questions / FAQ]
### [H3 — Question 1?]
### [H3 — Question 2?]

## [H2 — Section 6: Conclusion / Key Takeaways]
```

**Instructions:** The outline is the skeleton of the article. Every H2 should be a standalone topic that could be extracted by an AI search engine as a featured snippet. Use question-based headings where the search intent is a question (e.g. "Does microneedling regrow hair?" not "Microneedling effectiveness"). The first section must answer the core question immediately — no lengthy introductions. Do not use generic headings like "Introduction" or "Conclusion" — every heading should be specific and keyword-relevant. Include 5-8 H2s for a standard article, 8-12 for a pillar page.

## 5. Word Count Target

| Field | Value |
|-------|-------|
| **Target word count** | [range, e.g. 2,500-3,000 words] |
| **Competitor average** | [average word count of top 3 competitors] |
| **Content type benchmark** | [cluster article: 1,500-2,500 / pillar page: 3,000-5,000 / comparison: 2,000-3,000] |

**Instructions:** Word count should be based on what's currently ranking, not an arbitrary target. Calculate the average word count of the top 3 competitors and aim for within 20% of that average. Longer is not automatically better — a 2,000-word article that answers the question completely will outperform a 5,000-word article padded with filler. If competitors are all 1,500 words, target 1,500-1,800 — don't write 4,000 just because "longer content ranks better." Match the depth to the search intent.

## 7. Meta Title

| Field | Value |
|-------|-------|
| **Proposed meta title** | [under 60 characters, primary keyword near the front] |
| **Character count** | [exact count] |
| **Primary keyword position** | [first / middle / end] |

**Instructions:** The meta title is the clickable headline in search results. It must be under 60 characters (Google truncates at ~60). The primary keyword should appear as close to the front as possible. Include a compelling reason to click — a number, a year, or a benefit. Examples of good titles: "Microneedling for Hair Growth: Complete 2026 Guide" (49 chars), "What Is Parental Alienation? Signs, Effects & What to Do" (56 chars). Avoid clickbait or misleading titles.

## 8. Meta Description

| Field | Value |
|-------|-------|
| **Proposed meta description** | [150-160 characters, includes primary keyword and a reason to click] |
| **Character count** | [exact count] |
| **Primary keyword included** | [Yes / No] |
| **Call-to-action** | [what action or benefit does the description promise?] |

**Instructions:** The meta description appears below the title in search results. It must be 150-160 characters (Google truncates longer descriptions). Include the primary keyword naturally — Google bolds matching keywords in descriptions, improving click-through rate. End with a clear value proposition or call-to-action that differentiates from competitors. Do not duplicate the meta title. Do not use quotes or special characters that might get truncated.
